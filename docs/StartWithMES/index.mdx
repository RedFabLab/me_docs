## Авторизация 

Для входа в систему введите пару логин/пароль и нажмите кнопку **Авторизоваться**.

![Autorization](./img/Autorization.png)

Для просмотра пароля нажмите кнопку **глаз**.

## Главная страница 

После авторизации открывается главная страница, отображающая:

1. Строку меню c переходом в разделы:
   - **Настройки**
   - **Личный кабинет**
   - **Ошибки**\
      Открывает [окно ошибок](https://yt.omegafuture.ru/articles/RedFabMES-A-796/%D0%A3%D0%B2%D0%B5%D0%B4%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F-%D0%B8-%D0%BE%D1%88%D0%B8%D0%B1%D0%BA%D0%B8).
   -  **Уведомления**\
      Открывает [окно уведомлений](https://yt.omegafuture.ru/articles/RedFabMES-A-796/%D0%A3%D0%B2%D0%B5%D0%B4%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F-%D0%B8-%D0%BE%D1%88%D0%B8%D0%B1%D0%BA%D0%B8).
   -  **Обновить**\
      Выполняет перезагрузку страницы.
   -  **Выйти**\
      Пользователь выходит из системы. Открывается окно регистрации.
2. [**Уведомления и ошибки**](https://yt.omegafuture.ru/articles/RedFabMES-A-796/%D0%A3%D0%B2%D0%B5%D0%B4%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F-%D0%B8-%D0%BE%D1%88%D0%B8%D0%B1%D0%BA%D0%B8)
3. "Шапку" области принтера
   -  Название и операционную группу принтера
   -  Чек-бокс **Включить в очередь печати**\
      Включение чек-бокса помещает принтер в [очередь печати](https://yt.omegafuture.ru/articles/RedFabMES-A-788/%D0%9E%D1%87%D0%B5%D1%80%D0%B5%D0%B4%D1%8C-%D0%BF%D0%B5%D1%87%D0%B0%D1%82%D0%B8). Теперь на него будут назначаться печать копий задач.
   -  [Терминал](https://yt.omegafuture.ru/articles/RedFabMES-A-775)
4. [**Область принтера**](https://yt.omegafuture.ru/articles?draft=132-13288)

![MainPage](./img/MainPage.png)

## Настройки

- При включении чек-бокса **Сбрасывать бракованные детали автоматически** при браке детали система 
автоматически выполняет отпуск вакуума и протягивание детали, а также начинает новую печать копии;
- При включении чек-бокса **Генерировать отчеты об ошибках принтеров** система автоматически скачивает 
отчеты об ошибках принтера, возникших в ходе работы;
- При включении чек-бокса **Автоматически отправлять отчеты об ошибках в техническую поддержку** система 
автоматически будет отправлять отчет об ошибках специалистам технической поддержки.

Скачивание и отправка отчетов возможна и выборочно, для конкретной ошибки, подробности описаны         
в статье 
[Уведомления и ошибки](https://yt.omegafuture.ru/articles/RedFabMES-A-796/%D0%A3%D0%B2%D0%B5%D0%B4%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F-%D0%B8-%D0%BE%D1%88%D0%B8%D0%B1%D0%BA%D0%B8 "RedFabMES-A-796: Уведомления и ошибки").

![Properies](./img/Properies.png)

Кнопка **Перейти в Центр обновлений** открывает пользователю [страницу центра обновлений](https://yt.omegafuture.ru/articles/RedFabMES-A-797/%D0%A6%D0%B5%D0%BD%D1%82%D1%80-%D0%BE%D0%B1%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B9).

## Личный кабинет 

Для перехода в личный кабинет нажмите кнопку **Личный кабинет**. В личном кабинете доступны 
следующие операции:

1. Смена имени и фамилии
2. Сброс пароля
3. Сохранение введенных изменений
4. Просмотр доступного функционала (зависит от выданной [роли](https://yt.omegafuture.ru/articles/RedFabMES-A-774/%D0%A0%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D1%8F-%D0%BC%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C))

![LK](./img/LK.png)

При необходимости создания/удаления/редактирования учетных записей - рекомендуется ознакомиться 
с [разделом **Пользователи**](https://yt.omegafuture.ru/articles/RedFabMES-A-790/%D0%A0%D0%B0%D0%B1%D0%BE%D1%82%D0%B0-%D1%81%D0%BE-%D1%81%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B0%D0%BC%D0%B8#%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D0%B8).

## Запуск печати 

В MES-системе существует 3 способа запуска печати:

1. Через очередь печати (см. [Очередь печати](https://yt.omegafuture.ru/articles/RedFabMES-A-780/%D0%96%D0%B8%D0%B7%D0%BD%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9-%D1%86%D0%B8%D0%BA%D0%BB-%D0%B8-%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5-%D0%BF%D1%80%D0%BE%D0%B5%D0%BA%D1%82%D0%B0 "RedFabMES-A-780: Жизненный цикл и создание проекта"))
2. Через [**PRM**](https://yt.omegafuture.ru/articles/RedFabMES-A-793/PRM)
3. Через [**терминал**](https://yt.omegafuture.ru/articles/RedFabMES-A-775/%D0%A0%D0%B0%D0%B1%D0%BE%D1%82%D0%B0-%D1%81-%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D0%BB%D0%BE%D0%BC)
   При печати терез **терминал** копия задания не помещается в очередь печати. Печать через терминал возможна 
   только, если принтер находится [в статусе](https://yt.omegafuture.ru/articles/RedFabMES-A-779/%D0%9E%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C-%D0%BF%D1%80%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%B0#%D1%81%D1%82%D0%B0%D1%82%D1%83%D1%81%D1%8B-%D0%BF%D1%80%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BE%D0%B2) 
   **Готов к работе** и НЕ в очереди печати.\
   Для печати через терминал можно выбрать:
   -  **Печать из файла**\
      Для печати из файла необходимо нажать кнопку **Печать из файла**, а затем **Выбрать файл**. 
      Откроется окно проводника для выбора файла на вашем ПК. Поддерживается только формат gcode.
   -  **Печать задания**

![TaskPrinting](./img/TaskPrinting.png)

:::caution Внимание

      Это может привести к аварии!
