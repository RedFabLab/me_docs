## Авторизация 

Для входа в систему введите пару логин/пароль и нажмите кнопку **Авторизоваться**.

![Autorization](./img/Autorization.png)

Для просмотра пароля нажмите кнопку **глаз**.

## Главная страница 

После авторизации открывается главная страница, отображающая:

1. "Верхнее меню" c переходом в разделы:
   - **Настройки**
   - **Личный кабинет**
   - **Ошибки**\
      Открывает [**окно ошибок**](/docs/StartWithMES/NotificationsAndErrors/#ошибки).
   -  **Уведомления**\
      Открывает [**окно уведомлений**](/docs/StartWithMES/NotificationsAndErrors/#уведомления).
   -  **Обновить**\
      Выполняет перезагрузку страницы.
   -  **Выйти**\
      Пользователь выходит из системы. Открывается окно регистрации.
2. [**Уведомления и ошибки**](/docs/StartWithMES/NotificationsAndErrors/)
3. "Шапку" области принтера
   -  Название и операционную группу принтера
   -  Кнопка переключения **Включить в очередь печати**\
      Включение кнопки переключения помещает принтер в [**очередь печати**](/docs/StartWithMES/PrintQueue/PrintQueueRules/). 
      Теперь на него будет назначаться печать копий заданий.
   -  [**Терминал**](/docs/StartWithMES/WorkingWithTerminal)
4. [**Область принтера**](/docs/StartWithMES/PrinterArea)

![MainPage](./img/MainPage.png)

## Настройки

- При включении кнопки переключения **Сбрасывать бракованные детали автоматически** при браке детали система 
автоматически выполняет отпуск вакуума и протягивание детали, а также начинает новую печать копии;
- При включении кнопки переключения **Генерировать отчеты об ошибках принтеров** система автоматически генерирует 
отчеты об ошибках принтера, которые в дальнейшем могут быть отправлены в техподдержку;
- При включении кнопки переключения **Автоматически отправлять отчеты об ошибках в техническую поддержку** система 
автоматически будет отправлять отчет об ошибках специалистам технической поддержки.

Скачивание и отправка отчетов возможна и выборочно, для конкретной ошибки, подробности описаны в 
статье [**Уведомления и ошибки**](/docs/StartWithMES/NotificationsAndErrors/).

![Properies](./img/Properies.png)

Кнопка **Перейти в Центр обновлений** открывает пользователю 
страницу [**центра обновлений**](/docs/StartWithMES/UpdateCenter).

Кнопка **Скачать отчет** в разделе **Журнал событий** скачивает отчет обо всех непрочитанных ошибках и уведомлениях.
Для скачивания отчета о конкретной ошибке необходимо перейти в раздел 
[**история ошибок**](/docs/StartWithMES/NotificationsAndErrors/#история-ошибок).

## Личный кабинет 

Для перехода в личный кабинет нажмите кнопку **Личный кабинет**. В личном кабинете доступны 
следующие операции:

1. Смена имени и фамилии
2. Сброс пароля
3. Сохранение введенных изменений
4. Просмотр доступного функционала (зависит от выданной [**роли**](/docs/StartWithMES/RoleModel))

![LK](./img/LK.png)

При необходимости создания/удаления/редактирования учетных записей - рекомендуется ознакомиться 
с разделом [**Пользователи**](/docs/WorkingWithReferenceBooks#пользователи).

## Запуск печати 

В MES-системе существует 3 способа запуска печати:

1. Через [**очередь печати**](/docs/StartWithMES/PrintQueue/)
2. Через [**Слайсер**](/docs/Slicer/SlicerPrintingSettings#слайситьв-печатьскачать)
3. Через [**терминал**](/docs/StartWithMES/WorkingWithTerminal)\
   Печать через **терминал** печатает отдельный выбранный gcode и никак не связана с очередью печати и её 
   [**жизненным циклом**](/docs/StartWithMES/PrintQueue/#жизненный-цикл-проектов). Печать через терминал возможна 
   только, если принтер находится [**в статусе**](/docs/StartWithMES/PrinterArea#статусы-принтеров) 
   **Готов к работе** и НЕ в очереди печати.\
   Для печати через терминал можно выбрать:
   -  **Печать из файла**\
      Для печати из файла необходимо нажать кнопку **Печать из файла**, а затем **Выбрать файл**. 
      Откроется окно проводника для выбора файла на вашем ПК. Поддерживается только формат gcode. 
   -  **Недавние файлы**\
      Нажатие на кнопку **Недавние файлы** открывает список gcode, распечатаных ранее. При выборе необходимого gcode
      из открывшегося списка - система распечатает 3D-модель.
   -  **Печать задания**\
      Из выбраного задания выбирается gcode файл и печатается 1 модель (назависимо от колличетсва копий в задании).

![TaskPrinting](./img/TaskPrinting.png)

:::caution Внимание
      Будьте осторожны при запуске gcode файлов из неизвестных источников.
      Это может привести к аварии!
:::
