Ролевая модель в MES-системе нужна для оптимизации процессов. 
Она содержит набор разрешений, позволяющий выполнять определенные действия с ресурсами системы. 
С помощью ролевой модели\
вы управляете контролем доступа, определяя, кто имеет какой доступ (роль) для какого ресурса.

:::tip Пример
      Стажеру назначается ограниченная роль, чтобы предотвратить случайные ошибки, 
      которые могут повлиять на всю систему. В отличие от этого, сотрудник техподдержки имеет доступ 
      к критическим функциям, что позволяет эффективно сопровождать MES-систему и своевременно устранять проблемы.
:::
