---
title: Минимальные системные требования
sidebar_position: 1
---
Для исправной работы MES-системы ПК пользователя должен удовлетворять следующим системным требованиям:

- Процессор не хуже AMD Ryzen 5 3600 со встроенной графикой;
- Оперативная память - не менее 16Gb;
- Твердотельный накопитель SSD 1Tb (Возможно увеличение объема с возможностью зеркалирования RAID 1);
- OS Linux (Ubuntu Desktop 22.04 LTS);
- При удаленном управлении через ПК необходимо иметь (установить если отсутствует) один из следующих браузеров: 
Chrome, Firefox версии 124, Microsoft Edge версии 94, Opera версии 109, Safari версии 17, Yandex.browser версии 24.
